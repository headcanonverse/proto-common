# Правила (см. корневой ~/headcanonverse/CLAUDE.md — полная версия)

ЖЕЛЕЗНО: схема БД — ТОЛЬКО GORM-модели + AutoMigrate. Рукописные миграции/DDL/Migrator().Drop*/Add* ЗАПРЕЩЕНЫ. Индексы — тегами моделей. Убрать поле = удалить из модели; колонки-сироты не трогать. Обратная совместимость данных не нужна (не прод). Исключения (с комментарием): CREATE EXTENSION vector, pgvector-литералы/`<=>`.
Тесты: только фронтовые e2e Playwright; бэкенд — лишь офлайн-парсеры. Docker-сборка из корня репо; proto-common только по тегу.

Проблемы не скипать: «pre-existing» — не оправдание (тривиальное чинить при касании + gofmt -w, нетривиальное — эскалировать тикетом). E2e-тесты создают данные сами, seed-данные — запрещённая зависимость.
